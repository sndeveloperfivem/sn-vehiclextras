# sn-vehiclextras
Vehicle Appearance & Extras System for FiveM

A modern and lightweight vehicle customization system for FiveM, built with Lua, HTML, CSS, and JavaScript.
This script provides an in-game UI that allows players to customize vehicle liveries, chassis modifications, interior parts, and vehicle extras in real time.

Features a clean and compact design, smooth sliders for modifications, and instant application of changes without delays.
Fully compatible with GTA V vehicle mod indexes and designed to be easily expandable and customizable.

Features:

Vehicle livery (sticker) selector with live preview

Chassis and body modifications (bumpers, roof, exhaust, hood, etc.)

Interior modifications (seats, steering wheel, dashboard, gauges, etc.)

Vehicle extras toggle system

Real-time mod application

Modern, responsive, and minimal UI

Easy integration with existing FiveM resources
